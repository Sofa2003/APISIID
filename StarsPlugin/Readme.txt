Данный проект реализует функциональность приложения к T-FLEX CAD
В этом приложении создаются объекты с новым поведением (звёзды)
После компиляции плагин может быть запущен из диалога "Приложения..." T-FLEX CAD.

Для сборки может потребоваться обновить пути к зависимостям проекта:

  C:\Program Files\T-FLEX CAD 16\Program\TFlexAPI.dll
  C:\Program Files\T-FLEX CAD 16\Program\TFlexCommandAPI.dll

Свойство CopyLocal у зависимостей следует установить в False.
